# ⚙️ 01 - Arithmetic Operators

> **Module 02 • Operators**

Arithmetic operators are among the most fundamental operators in Python. They are used to perform mathematical calculations such as addition, subtraction, multiplication, division, and more. Whether you're building a calculator, a billing system, a banking application, or a data analysis project, arithmetic operators are used extensively.

This lesson introduces each arithmetic operator with clear explanations, practical examples, and a real-world mini project to strengthen your understanding.

# 📚 Operators Covered

| Operator | Description | Example |
|----------|-------------|---------|
| `+` | Addition | `10 + 5` |
| `-` | Subtraction | `10 - 5` |
| `*` | Multiplication | `10 * 5` |
| `/` | Division | `10 / 5` |
| `//` | Floor Division | `10 // 3` |
| `%` | Modulus (Remainder) | `10 % 3` |
| `**` | Exponentiation | `2 ** 4` |

# 📌 Key Takeaways

- Arithmetic operators perform mathematical calculations.
- Python provides seven built-in arithmetic operators.
- `/` always returns a floating-point result.
- `//` returns the integer quotient (floor value).
- `%` returns the remainder after division.
- `**` calculates the power of a number.
- Arithmetic operators are fundamental to almost every Python application.
