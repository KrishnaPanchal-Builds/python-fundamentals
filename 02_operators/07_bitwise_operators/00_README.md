# 💡 05 - Bitwise Operators

> **Module 02 • Operators**

Bitwise operators are used to perform operations directly on the binary (bit-level) representation of integers. Instead of working with decimal values, these operators manipulate individual bits, making them useful in areas such as system programming, networking, data compression, encryption, embedded systems, and performance optimization.

Bitwise operators may seem advanced at first, but understanding their fundamentals provides a strong foundation for low-level programming and efficient data manipulation.

# 📚 Operators Covered

| Operator | Description | Example |
|----------|-------------|---------|
| `&` | Bitwise AND | `5 & 3` |
| `\|` | Bitwise OR | `5 \| 3` |
| `^` | Bitwise XOR | `5 ^ 3` |
| `~` | Bitwise NOT | `~5` |
| `<<` | Left Shift | `5 << 1` |
| `>>` | Right Shift | `5 >> 1` |

# 🔢 Understanding Binary Numbers

Computers store integers in binary form.

Example:

| Decimal | Binary |
|----------|---------|
| 1 | 0001 |
| 2 | 0010 |
| 3 | 0011 |
| 4 | 0100 |
| 5 | 0101 |
| 6 | 0110 |
| 7 | 0111 |
| 8 | 1000 |

Bitwise operators perform operations directly on these binary values.

# 📌 Key Takeaways

- Bitwise operators work directly on binary values.
- They operate on integers only.
- They are useful for low-level and performance-critical programming.
- Left Shift multiplies by powers of two.
- Right Shift divides by powers of two (for positive integers).
- Bitwise operators are widely used in systems programming and networking.
