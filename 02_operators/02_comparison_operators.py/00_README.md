# ⚖️ 02 - Comparison Operators

> **Module 02 • Operators**

Comparison operators are used to compare two values or expressions in Python. They always return a Boolean value (`True` or `False`), making them the foundation of decision-making in programming.

Although comparison operators are simple, they play a crucial role in conditions, loops, data validation, searching, filtering, and many real-world applications.

# 📚 Operators Covered

| Operator | Description | Example |
|----------|-------------|---------|
| `==` | Equal To | `10 == 10` |
| `!=` | Not Equal To | `10 != 5` |
| `>` | Greater Than | `20 > 15` |
| `<` | Less Than | `8 < 12` |
| `>=` | Greater Than or Equal To | `18 >= 18` |
| `<=` | Less Than or Equal To | `15 <= 20` |
