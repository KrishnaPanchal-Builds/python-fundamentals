# 📝 04 - Assignment Operators

> **Module 02 • Operators**

Assignment operators are used to assign values to variables and update those values efficiently. They make Python code shorter, cleaner, and easier to read by combining an operation with assignment into a single statement.

These operators are commonly used when working with counters, financial calculations, inventory management, scores, statistics, and many other real-world applications.

# 📚 Operators Covered

| Operator | Description | Example |
|----------|-------------|---------|
| `=` | Assigns a value | `salary = 50000` |
| `+=` | Adds and assigns | `salary += 5000` |
| `-=` | Subtracts and assigns | `salary -= 2000` |
| `*=` | Multiplies and assigns | `salary *= 2` |
| `/=` | Divides and assigns | `salary /= 2` |
| `//=` | Floor divides and assigns | `salary //= 2` |
| `%=` | Finds remainder and assigns | `salary %= 1000` |
| `**=` | Raises to a power and assigns | `salary **= 2` |
