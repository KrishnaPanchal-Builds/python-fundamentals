# 📖 Overview

Welcome to **Module 01** of the **Python Fundamentals** repository.

This module introduces the core concepts of Python programming and builds the foundation required for every topic that follows. You'll learn how to create variables, work with different data types, accept user input, manipulate strings, perform numerical calculations, and understand Boolean values.

Each lesson is carefully designed with practical examples to help you develop confidence in writing clean, readable, and efficient Python code.

---

# 🎯 Learning Objectives

After completing this module, you will be able to:

- 📝 Create and manage variables
- 🔢 Understand Python data types
- 🔄 Convert values using type casting
- ⌨️ Accept and process user input
- ✍️ Perform string manipulation
- ➕ Work with numbers and mathematical expressions
- ✅ Understand Boolean values and expressions
- 🧠 Develop logical thinking and problem-solving skills
